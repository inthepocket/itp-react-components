# ITP Slider component
## Usage
```
$ npm install @inthepocket/itp-rcc-slider
```

Import (Typescript)
```javascript
import { default as Slider } from '@inthepocket/itp-rcc-slider';
```

Import (ES)
```javascript
import Slider from '@inthepocket/itp-rcc-slider/dist/Slider';
```

JSX
```jsx
<Slider />
```
